## Clone Repository

```sh
git clone https://github.com/friccardi310/smooth-scroll-to-top
```

## Change Directory

```sh
cd smooth-scroll-to-top
cd demo
```

## Package Installation

```sh
npm install
```

Or using yarn:
```sh
yarn add smooth-scroll-to-top
```

## Run the App

```sh
npm run
```
